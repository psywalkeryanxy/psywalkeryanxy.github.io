This is a companion post to the paper. The published version contains the formal results; here I’ll write more freely about how we got there, what surprised us, and what the analysis didn’t tell us.

The question

Bilingual speakers reach the same concept (“dog”, “perro”) through two very different surface forms. Where does that convergence happen, and what does it look like at the population level?

The finding in one sentence

English and Spanish nouns are encoded along nearly orthogonal axes in human hippocampal population activity (≈85° principal angles), yet the geometric relationships between concepts are preserved across languages — which we summarize as shared geometry, rotated readout axes.

Why I think this matters

Most work on bilingual semantic memory has either looked for “shared neurons” (translation neurons) or for fully separate representations. The geometric framing dissolves that dichotomy: the same relational structure can live in two different subspaces. That’s also a clean prediction from how you’d expect a high-dimensional code to multiplex.
