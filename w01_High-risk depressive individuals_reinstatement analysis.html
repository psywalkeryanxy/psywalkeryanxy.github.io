Hamming window¶

Preprocessed resting time series convolved with hamming window (left figure), three-point-width hamming window whose shape most like HRF, if the parameter used 5, 6, 7 but not 3, the shape wont be like with HRF (right figure). Then transferred the convolved signal to percent signal change.

materials = ['3selfMFG_10--18_30_33']
x_pos = np.arange(len(materials))
CTEs = [actual_mean]
error = [actual_se]


fig, ax = plt.subplots()
ax.bar(x_pos, CTEs, yerr=error, align='center', alpha=0.5, ecolor='black', capsize=10)
ax.set_ylabel('Coefficient across all subs')
ax.set_xticks(x_pos)
ax.set_xticklabels(materials)
ax.set_title('reinstatement value, p = 0.0202*')
ax.yaxis.grid(False)
